# Animation Game Challenge 🎮

This is a fun and interactive web-based animation project built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
http://13.239.227.75

## ☁️ Deployment
The project is deployed on AWS using:
- EC2 (for hosting)
- S3 (for storing project files)

## 💡 Features
- Smooth animations
- Interactive UI
- Includes images and audio
- Beginner-friendly game concept
